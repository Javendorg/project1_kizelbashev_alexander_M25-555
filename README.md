# Лабиринт сокровищ (labyrinth_game)

    Небольшая текстовая игра-лабиринт на Python. Игрок перемещается по комнатам, подбирает предметы, решает головоломки и пытается добраться до комнаты с сокровищами.

## Ключевые особенности:
- Простая управляемая логика перемещения по комнатам
- Инвентарь игрока и использование предметов
- Загадки/головоломки в отдельных комнатах
- Возможность расширения набора комнат и предметов

## Установка
1. Рекомендованный (Poetry):

```bash
poetry install
```

2. Через Make:

```bash
make install
```

## Как запустить

1. Через консоль Poetry:

```bash
poetry run project
```

2. Через Make:

```bash
make project
```
## Как победить
- При запуске вы попадаете в начальную комнату (entrance).
- Перемещение: north, south, east, west или `go <direction>` (например `go north`).
- Команды:
	- `look` — осмотреть текущую комнату
	- `take <item>` — взять предмет
	- `use <item>` — использовать предмет из инвентаря
	- `inventory` — показать инвентарь
	- `solve` — попытаться решить головоломку в комнате
	- `help` — вывести подсказки по командам
	- `quit` / `exit` — выйти из игры

## Как победить
    Цель игры — открыть сундук в `treasure_room`. Классический путь к победе выглядит так:

    1. Найдите способ добраться до `treasure_room`. Дверь к комнате сокровищ заперта — чтобы пройти в неё, обычно нужен ключ `rusty_key` (например, он лежит в `trap_room`, или можно получить его, открыв `bronze_box` в `armory`).
    2. Попав в `treasure_room`, откройте сундук. Есть два рабочих способа:
        - Использовать предмет `treasure_key`, если вы его получили раньше — тогда сундук откроется автоматически.
        - Ввести правильный код при запросе. Подсказка к коду есть в описании комнаты — код равен `10`.
